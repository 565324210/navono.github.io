我们从以下几个方面对shared_ptr对象和裸指针进行比较。

缺失 ++，– 和 [] 操作符

和裸指针相比，shared_ptr对象只提供以下操作符：

  • ->, *
  • 比较操作符

不提供以下：

  • 算数运算，比如：+，-，++，–
  • []操作符

NULL 检查
